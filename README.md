# muskanahuja.com

Personal portfolio site for Muskan Ahuja — Financial Analyst based in Richmond, VA.

Built with Next.js 14, Tailwind CSS, and Framer Motion.

## Local development

```bash
pnpm install
pnpm dev
```

Site runs at `http://localhost:3000`.

## Deployment

Auto-deployed to Vercel on push to `master`. Live at [muskanahuja.com](https://muskanahuja.com).
